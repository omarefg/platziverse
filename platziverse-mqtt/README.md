# platziverse-mqtt

## `agent/connected`

```js
{
    agent: {
        uuid, // autogenerated
        username // defined by configuration
        name, // defined by configuration
        hostname, // getted by os
        pid // getted by process
    }
}
```

## `agent/disconnected`

```js
{
    agent: {
        uuid
    }
}
```

## `agent/message`

```js
{
    agent,
    metrics: [
        {
            type,
            value
        }
    ],
    timestamp // generated with the message,
}
```
